# quiz-app

一般常識クイズアプリ。HTML/CSS/JavaScriptのみで構成されたシンプルなフロントエンドアプリ。

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript (フレームワークなし)

## ファイル構成

- `index.html` — アプリの構造。クイズ画面と結果画面の2セクションを持つ
- `style.css` — スタイル定義
- `script.js` — クイズロジック、問題データ、DOM操作

## アプリの仕様

- 問題数: 5問（`script.js` の `questions` 配列で管理）
- 各問題は4択形式
- 回答後に正解・不正解のフィードバックと解説を表示
- 全問終了後にスコアと結果コメントを表示
- 「もう一度挑戦する」ボタンでリセット可能

## 問題の追加・編集

`script.js` の `questions` 配列にオブジェクトを追加する:

```js
{
  text: "問題文",
  choices: ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  correct: 0,        // 正解のインデックス（0始まり）
  explanation: "解説文"
}
```

## 注意事項

- サーバー不要。`index.html` をブラウザで直接開いて動作確認できる
- 外部ライブラリ・CDN依存なし
